# Acc102-Track4
With the development of the internet, the e-commerce economy is booming. Many people want to run their own online stores, yet struggle to decide what products to source and who their target customers should be. This tool is tailor-made for aspiring e-commerce sellers who fail to identify their target audience and select sellable goods.

Data acquisition：The dataset is downloaded from Kaggle using Python and stored in the local project folder.Data cleaning and preprocessing：Missing values in key fields are removed, dates are converted into datetime format, and monthly transaction periods are generated using . High-frequency user identification.Predictive scoring and recommendation logic.Interactive visualization and comparison.

1.Customers with higher disposable income and lower cancellation rates tend to have a higher estimated purchase success rate.2.High-frequency customers are more likely to be stable and valuable target groups for e-commerce marketing.3.Monthly expense alone is not enough to judge purchasing power; the balance between income, expense, and cancellation behavior is more informative.
